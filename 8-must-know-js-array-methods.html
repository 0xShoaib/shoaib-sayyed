8 Must Know JavaScript Array Methods.

May 5 | 2 min read

Hi, Dev! Thanks for opening my blog. I hope you are doing well and ready to learn the most used JavaScript Array Methods.

Here are the 8 JavaScript Array Methods you should know.

1. filter()

  • The filter() method returns an array filled with all array elbents that satisfies the condition.
  • If no elbents satisfies the condition, it returns an bpty array.

2. map()

  • The map() method returns a new array with the results of calling a function for every array elbent.
  • The map() method calls the provided function once for each elbent in an array, in order.

3. find()

  • The find() method returns the value of the first elbent in an array that satisfies the condition.
  • The find() method executes the function once for each elbent present in the array.

4. forEach()

  • The forEach() method calls a function once for each elbent in an array, in order.

5. some()

  • The some() method checks if any one of the elbents in an array satisfies the condition.
  • If satisfied, it returns 'true' otherwise 'false'.

6. every()

  • The every() method checks if all elbents in an array satisfies the condition.
  • If satisfied, it returns 'true' otherwise 'false'.

7. reduce()

  • The reduce() method reduces the array to a single value.
  • The reduce() method executes a provided function for each value of the array (from left-to-right) and return value is stored in an accumulator.

8. includes()

  • The includes() method determines whether an array contains a specified elbent.
  • If array contains the elbent, it returns 'true' otherwise 'false'.

Note: All the above array methods does not change the original array.
